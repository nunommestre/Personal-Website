Fibonacci Photo Encrypter

A simple command line project that encrypts an image using a left Fibonacci shift register with respect to the 16 bit argument passed in. The only way to decrypt it is to use the exam same key when re-running the program.

Evil Hangman

An evil hangman game in the command line that is constantly changing the winning word by using binary trees. Additionally I created a my_string library to handle character arrays in C. Feeling lucky? The project is available for download on Github.

Sierpinski Trangle

Ants and DoodleBugs

A predator vs prey simulation that starts with 5 red dots and 100 green dots. The 5 dots serve as predators hunting down the 100 prey. Both organisms both have very different reproductive systems that trigger at the same time with polymorphism. Can they catch all the prey before they starve out?
